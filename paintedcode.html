Auguste Herbin's Lion in Code

This is my recreation of Auguste Herbin's Lion. Herbin is a geometric abstraction artist. He created this piece in 1947 using gouache on paper. Gouache is a type of painting that uses opaque pigments that are thickened with a glue substance. My recreation is made completely out of HTML and CSS code.
